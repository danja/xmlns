GRDDL Profile for Links

Extracts RDF data for links in HTML documents

Profile URI : http://purl.org/stuff/glink/

Use of this profile licenses RDF data extracted by groklinks.xsl

Status : experimental (2007-10-29)

Usage

Insert the following into the head element in your XHTML document:

 ...
   <head profile="http://purl.org/stuff/glink/">
 ...

(note that multiple profile URIs can appear in the attribute, space-separated)
