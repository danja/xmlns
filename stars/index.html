Linked Open Data Ratings

The material is -

  1. ★ available on the Web (any format) under an open license
  2. ★★ available as structured data (e.g., Excel instead of image scan of a table)
  3. ★★★ uses non-proprietary formats (e.g., CSV instead of Excel)
  4. ★★★★ uses URIs to identify things, so that people can point at your stuff
  5. ★★★★★link your data to other data to provide context

For further examples and reasons for publishing 5 Star data, see 5stardata.info

See also the 5 stars of open linked data

Vocabulary Ratings

The vocabulary -

  1. ★ is published on the Web at a stable URI.
  2. ★★ has human-readable documentation and basic metadata such as creator, publisher, date of creation, last modification, version number is provided.
  3. ★★★ has labels and descriptions provided in several languages, to make your vocabulary usable in multiple linguistic scopes.
  4. ★★★★ is available via its namespace URI, both as a formal file and human-readable documentation, using content negotiation.
  5. ★★★★★ includes links to related vocabularies and wherever possible elements are reused rather than re-invented.

To encourage reuse, all vocabularies should specify an open license.

See also : Best Practice Recipes for Publishing RDF Vocabularies
