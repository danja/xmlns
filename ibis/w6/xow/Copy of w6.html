XOW - XHTML Outlines for W6

2004-02-22 - work-in-progress

The following is a first bash at an XHTML representation of w6. It's an expandable outline, so clicking on the grey squares should expand/contract it. I'm working in IE, but I'll try and keep it working in Mozilla, it should fall back on being a simple outline list in Lynx etc. Be sure and view the source. There's an XSLT stylesheet to generate the RDF (after pretty-printing), and another (xow2bookmarks.xsl) that puts the data in a format suitable for importing into Mozilla/Firefox (w6bm.html). I'm working on making this in-place editable, so it can be used for rapid input/annotation. Lots to do...

See also: XHTML Outlines and W6 notes on my blog, also XO and GRDDL
